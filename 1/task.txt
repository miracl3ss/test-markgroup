Описание: Написать простое веб-приложение на JavaScript, которое отображает список задач (To-Do List). Приложение должно позволять добавлять новые задачи, помечать задачи как выполненные и удалять задачи.

Требования:
Реализовать интерфейс без использования фреймворков (только HTML, CSS и чистый JavaScript).
Задачи должны сохраняться в localStorage, чтобы оставаться на странице при перезагрузке.
Код должен быть хорошо структурирован и понятен.

Цель: Оценить базовые знания JavaScript, умение работать с DOM и localStorage, а также умение писать структурированный код.
